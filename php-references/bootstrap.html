Übung Bootstrap

Erstelle eine Seite rein nur mit Bootstrap Klassen.

Binde die Online-Ressource von Bootstrap ein (damit du nicht in die Versuchung kommst die vordefinierten Klassen umzuschreiben)

Du kannst Bilder über Stockfoto Anbieter benutzen sowie Lorem Ipsum Texte.

Du darfst KEIN eigenes CSS schreiben! Arbeite rein nur mit den vorgegebenen Klassen und achte darauf dass die Seite responsive ist.

Die Seite soll beinhalten:

  • Carousel
  • Collapse Element
  • responsive Tabelle
  • Buttons mit verschiedenen Farben
  • einen Bereich mit Card Layout
  • Progress Bar
